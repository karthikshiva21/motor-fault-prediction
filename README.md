# Motor Fault Prediction using Machine Learning

## 📌 Project Overview
This project predicts motor faults using temperature and vibration data.

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 🤖 Model Used
- Logistic Regression

## 📊 Features
- Predicts whether motor is Normal (0) or Fault (1)
- Uses sensor data (temperature & vibration)
- Includes data visualization

## 🧠 ML Workflow
- Data collection
- Data preprocessing
- Train-test split
- Model training
- Prediction
- Accuracy evaluation

- Use real industrial dataset
- Try Decision Tree & Random Forest
